what are props?
props=properties
used to pass data from parent to child
props always are READ-ONLY
think props like your funtion parametes

example:
teacher gives instructions to students

teacher->parent component
students->child component
instructions->props(data)

3 ways of conditional rendering
1. if else
example:
if(isLoggedIn){
    return <h1> welcome</h1>

}else{
    return <h1> please Login</h1>
}

2. Ternary Operator
{isLoggedIn ? <h1> welcome</h1> : <h1> please Login </h1>}

3. Logical And(&&)
{isLoggedIn && <h1> welcome</h1>}
